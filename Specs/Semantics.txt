Variables:
+Variables Always Declared before Use
+Variables should never be ReDeclared within same scope
+Variables Should Never have void type
+Const Variables should never be changed

Functions:
+Functions should Never be ReDeclared within Same Scope
+Cannot Return from Void Functions
+Functions with non-void return types must always return data
+Functions Can Contain Only 1 Return Statement in 1 Scope
&Call Undeclared Functions Error
-Number of FunctionParameters/Arguments Passed
-Parameters with Default Value

Statements:
ControlStatement and IterationStatement Bodies are Optional
